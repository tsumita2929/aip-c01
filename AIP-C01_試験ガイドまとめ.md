# AWS Certified Generative AI Developer - Professional (AIP-C01) 試験ガイドまとめ

## 試験概要

| 項目           | 内容                                             |
| -------------- | ------------------------------------------------ |
| 試験コード     | AIP-C01                                          |
| カテゴリ       | Professional                                     |
| 試験時間       | 205分                                            |
| 総問題数       | 85問（採点対象65問 + 採点対象外10問）            |
| 設問形式       | 択一選択、複数選択、並べ替え、内容一致           |
| 合格スコア     | 750 / 1,000                                      |
| 受験料         | 150 USD                                          |
| 受験方法       | ピアソンVUE テストセンター or オンライン監督付き |
| 対象言語       | 英語・日本語                                     |
| 推測ペナルティ | なし（未解答は不正解扱い）                       |

## 受験対象者の推奨経験

- AWS またはオープンソース技術での**プロダクショングレードアプリケーション構築 2年以上**
- 一般的な AI/ML またはデータエンジニアリングの経験
- **GenAI ソリューション実装 1年**の実務経験

### 推奨される AWS 知識

- コンピューティング、ストレージ、ネットワークサービス
- セキュリティのベストプラクティスとアイデンティティ管理
- デプロイと Infrastructure as Code (IaC) ツール
- モニタリングおよびオブザーバビリティサービス
- コスト最適化の原則

### 試験対象外のジョブタスク

- モデルの開発とトレーニング
- 高度な ML 手法
- データエンジニアリングと特徴量エンジニアリング

## 模擬問題作成のための重点ポイント

### 分野1（31%）で特に重要なトピック

3. **プロンプトエンジニアリング**: Prompt Management、Prompt Flows、プロンプトチェーン、思考連鎖、ガバナンス
4. **MCP (Model Context Protocol)**: エージェント・ツール間の標準化されたインタラクション

4. **デプロイ戦略**: オンデマンド vs プロビジョンド、モデルカスケード

### 分野3（20%）で特に重要なトピック

4. **ガバナンス**: モデルカード、データリネージュ、監査ログ、ドリフト検出

### 分野4（12%）で特に重要なトピック

4. **モニタリング**: CloudWatch メトリクス、Bedrock モデル呼び出しログ、コスト異常検出
